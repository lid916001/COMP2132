# Hangman Game
## Project Structure
- `index.html` - Main game page
- `css/styles.scss` - SASS source file
- `css/styles.css` - Compiled CSS
- `js/game.js` - Game logic
- `data/words.json` - Words and hints
- `images/` - Hangman images (hangman-0.jpg to hangman-6.jpg)
